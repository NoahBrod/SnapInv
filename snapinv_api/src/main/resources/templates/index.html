Features
↧

Multiscan

Change quantity of item by scanning the same code multiple times on the fly. Scanned code not found? It will create a new item with that code.

Search and Filters

Customizable search for the name, id, and barcode of an item. Change filters for to list the items in your invenotry easier.

Graph Tracking

Track the stock of items on a customizable chart. Sets stock thresholds and get an alert when they are hit.

Offline Capability

Use the app while offline with most functionality available. Inventory will be updated once app is back online.
